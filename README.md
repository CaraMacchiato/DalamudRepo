Custom [XIVLauncher](https://github.com/goatcorp/FFXIVQuickLauncher) plugin repository.

## Installation

[![Dalamud custom repository setup](https://user-images.githubusercontent.com/12263784/212503458-8476c1e4-f09e-43a6-9da7-53fb2ae44dc6.png)](https://user-images.githubusercontent.com/12263784/212503458-8476c1e4-f09e-43a6-9da7-53fb2ae44dc6.png)

First open the Dalamud Settings menu in-game, either by typing `/xlsettings` in chat or by clicking the button at the bottom of Plugin Installer.

Then follow these steps:

1. Open the **Experimental** tab.
2. Paste `https://raw.githubusercontent.com/CaraMacchiato/DalamudRepo/main/repo.json` into an empty field under **Custom Plugin Repositories**.
3. Click the `+` button.
4. Click the save button.

After that, the plugins from this repository will appear in Plugin Installer (open with `/xlplugins`).
